# Bem-vindo à Documentação Go

> **Go** (ou Golang) é uma linguagem de programação open-source criada pelo Google em 2009, projetada para ser simples, eficiente e confiável.

## Por que aprender Go?

| Característica     | Descrição                                              |
|--------------------|--------------------------------------------------------|
| ⚡ Performance      | Compilada para binário nativo, próxima de C/C++        |
| 🧹 Simplicidade     | Sintaxe enxuta, sem hierarquias complexas              |
| 🔀 Concorrência     | Goroutines e Channels nativos na linguagem             |
| 🔒 Segurança        | Tipagem estática e garbage collector                   |
| 📦 Ecossistema      | Go Modules para gerenciamento de dependências          |
